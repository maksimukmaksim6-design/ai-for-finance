# PortfolioView — AI for Finance

An AI-powered portfolio analysis website built with Flask and LightGBM.

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/maksimukmaksim6-design/ai-for-finance
cd ai-for-finance
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the server
```bash
python server.py
```

### 4. Open the website
Go to [http://localhost:3000](http://localhost:3000) in your browser.
